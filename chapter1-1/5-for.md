# 문제 5 : for문 계산

### 문제 5 : for문 계산

다음 코드의 출력 값으로 알맞은 것은 ?

```javascript
let a = 10, b = 2;

for (let i = 1; i < 5; i += 2) {
  a += i;
}
console.log(a + b);
```

1. 10
2. 12
3. 14
4. 16

### 문제 5:  for문 계산\(풀이\)

#### for

for 문은 괄호로 감싸고 세미콜론으로 구분한 세 가지의 작업을 표현식으로 명시하고 다음과 같이 루프에 대한 문법을 정의해놨다.

```javascript
for ([초기문]; [조건문]; [증감문]) { ... }
```
